# 🌍 アジアンネットショップラマニ B2販売管理システム

yamato-b2-appはElectronアプリでFacebook Messenger等での注文情報を一元管理し、ヤマト運輸の送り状発行（B2クラウド連携）と納品書PDF作成を自動化するデスクトップアプリケーションです。

## 📌 目的

- **注文管理の効率化**: 散らばりがちなSNS経由の注文情報を一箇所に集約。
- **事務作業の自動化**: 手書きや手動入力によるミスを減らし、ヤマトB2用CSVと納品書PDFを即座に生成。
- **売上の可視化**: 当日・当月の出荷件数と売上金額をリアルタイムで把握。

## 🚀 使い方

### 1. 準備
- Node.jsがインストールされていることを確認してください。
- リポジトリをクローンまたはダウンロードします。
- ターミナルで以下のコマンドを実行して依存関係をインストールします。
  ```bash
  npm install
  ```

### 2. 起動
- 以下のコマンドでアプリケーションを起動します。
  ```bash
  npm run dev
  ```
cd ~/Projects/yamato-b2-app
npm run dev

### 3. 基本操作
1. **新規入力**: 「📝 新規入力」タブで顧客情報と商品情報を入力し「確認」をクリック。
2. **納品書発行**: 保存と同時に納品書PDFの保存ダイアログが表示されるので、任意の場所に保存します。
3. **履歴確認**: 「📋 履歴」タブで過去の注文を検索・確認できます。
4. **CSV出力**: 「📤 CSV出力」タブから期間を指定して、ヤマトB2クラウドにアップロード可能なCSVファイルを出力します。

## ⚙️ 仕様

### 技術スタック
- **基盤**: Electron (Desktop App)
- **データベース**: SQLite3 (ローカルストレージ)
- **帳票生成**: PDFKit (納品書PDF)
- **データ出力**: csv-writer (ヤマトB2標準フォーマット対応)

### 主要機能
- **ダッシュボード**: 今日の出荷、今日の売上、今月の出荷、今月の売上の統計表示。
- **入力フォーム**: 
  - 郵便番号からの住所入力補助（※手動入力）。
  - 自動計算機能（小計、消費税10%、合計金額）。
  - 配送方法の選択（発払い / コレクト・代金引換）。
- **データ連携**:
  - ヤマトB2用CSV出力: そのままヤマト運輸のB2クラウドへ取り込み可能。
  - 全項目CSV出力: 売上分析やバックアップ用の詳細データ出力。

### データの保存先
- アプリケーションのユーザーデータディレクトリ内の `sales.db` に保存されます。
